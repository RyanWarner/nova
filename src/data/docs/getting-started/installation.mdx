#### Create a repo from a template

The fastest way to start a new project is to use the [GitHub Template](https://help.github.com/en/github/creating-cloning-and-archiving-repositories/creating-a-repository-from-a-template) feature.

1. Navigate to the [Nova repository](https://github.com/RyanWarner/nova)
1. Click "Use this template"
1. Configure your repo and click "Create repository from template"

<br />

#### Running locally

Next, clone your newly created repository, install dependencies, and run the local server.

1. `git clone git@github.com:<username>/<repo>.git`
1. `cd <repo>`
1. `npm install`
1. `npm start`

By default, the app will open in your default browser at [http://localhost:8080](http://localhost:8080). This can be figured in `src/scripts/start.js`.
